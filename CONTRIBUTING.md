 # Contribuer à GameDevHub

Merci d’enrichir la communauté ! Voici comment procéder.

## Types de contributions
- **Article** (tutoriel, analyse technique)
- **Expérience** (retour, post-mortem)
- **Ressource** (outil, framework, guide)

## Où déposer ?
Créez un fichier **Markdown** dans le dossier `/contributions/` :
- Nom de fichier recommandé : `AAAA-MM-JJ-titre-court.md`
- Exemple : `2025-11-12-introduction-pixel-art.md`

Utilisez le modèle : `/contributions/_template-contribution.md`.

## Processus Git (fork + PR)
1. Forkez le dépôt.
2. Créez votre branche :
   ```bash
   git checkout -b ajout-contribution
